# Git - Commit Message

## Commit Type

###### add: 새로운 기능 추가

###### fix: 버그 수정

###### modify: 문서 수정

###### remove: 문서 삭제 및 기능 삭제

###### style: 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우

###### refactor: 코드 리펙토링

###### test: 테스트 코드

## Example

```
[add] 지도 api 추가
```
